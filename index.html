로컬 네트워크 권장
동일한 네트워크에 연결하면 모든 기능을 이용할 수 있어요.
  • 모든 기기를 동일한 Wi‑Fi에 연결해주세요.
  • VPN/사내 보안망이 켜져 있으면 연결이 안 될 수 있어요.
  • 연결이 안 되면 호스트의 핫스팟에 연결 후 앱을 새로고침해주세요.
이렇게 연결해요
방장이 알려주는 6자리 코드를 입력해 연결해요.
  • 방장: “제가 방장할래요” → 코드 확인 → “시작할래요!”
  • 참가자: “모임에 참여할래요” → 코드 입력 → 역할 선택(원본/왼쪽/오른쪽/저음)
  • 연결할 수 있는 기기는 방장 제외 최대 3대예요.
스피커로 재생하기
참가자가 역할(출력 채널)을 선택해요.
  • 중앙 스피커: 스테레오(기본) 출력
  • 왼쪽 스피커: L 채널 출력
  • 오른쪽 스피커: R 채널 출력
  • 서브우퍼: 저역 믹스 출력
필요하면 설정에서 역할을 언제든 바꿀 수 있어요.
미디어 재생하기
방장에게는 3가지 선택지가 나와요.
  • 로컬파일 불러오기: 기기 파일에서 음악/영상을 선택
  • 유튜브(채널분리 미지원): 링크를 붙여넣어 재생 목록에 추가
  • 앱 체험하기: 데모 미디어로 프로그램 테스트
도움이 필요할 때
  • 코드를 입력했는데 연결이 안 돼요: 서버 오류일 수 있어요. 이런 경우 두 기기를 같은 네트워크에 연결해주세요.
  • 연결이 불안정해요: 네트워크 품질이 낮을 수 있어요. 공유기 가까이로 이동해 보세요.
